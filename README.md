## 📖 About

This repository serves as a structured, progressive learning path for Playwright test automation. It starts with the fundamentals of JavaScript and Java, then gradually moves into Playwright-specific concepts, best practices, and real-world testing patterns.

Whether you're a beginner or transitioning from another automation tool, this repo is designed to guide you step by step.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/sandhyakrsna999/Learning_Playwright1.git
   ```

2. **Navigate into the project**
   ```bash
   cd Learning_Playwright1
   ```

3. **Verify your Node.js setup**
   ```bash
   node chapter_01_Basics/03_verify_setup.js
   ```

## 📘 Chapters

### Chapter 01 — JavaScript Basics

This chapter covers the foundational JavaScript concepts required before diving into Playwright:

| File | Topic | Description |
|------|-------|-------------|
